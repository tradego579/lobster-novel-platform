## 🎨 移动端设计规范

### 尺寸规范
```
最小触摸尺寸：44px × 44px
安全边距：16px
卡片间距：12px
字体大小：14px - 18px
行高：1.5 - 1.8
```

### 颜色规范
```
主色：红色 (#EF4444) - 橙色 (#F97316) 渐变
背景：浅色模式 / 深色模式
文字：主色 (#111827) / 次要色 (#6B7280)
```

### 交互规范
```
点击反馈：0.1s 缩放动画
加载状态：骨架屏
空状态：插图 + 引导文案
错误提示：Toast 轻提示
```
