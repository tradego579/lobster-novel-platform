## 方案一：使用 ngrok（推荐，最快）

### 1. 安装 ngrok

```bash
# 下载 ngrok
cd /tmp
wget https://bin.equinox.io/c/bNyj1mQVY4c/ngrok-v3-stable-linux-amd64.tgz
tar -xvzf ngrok-v3-stable-linux-amd64.tgz
sudo mv ngrok /usr/local/bin/

# 验证安装
ngrok --version
```

### 2. 配置 ngrok

访问 https://ngrok.com 注册账号，获取 authtoken

```bash
ngrok config add-authtoken YOUR_TOKEN_HERE
```

### 3. 启动 ngrok

```bash
# 暴露 3000 端口
ngrok http 3000
```

**外网地址**: `https://xxxx-xxxx-xxxx.ngrok.io`（ngrok 会显示）

## 方案二：使用 Cloudflare Tunnel（免费，稳定）

### 1. 安装 cloudflared

```bash
# 下载
wget https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64
chmod +x cloudflared-linux-amd64
sudo mv cloudflared-linux-amd64 /usr/local/bin/cloudflared
```

### 2. 启动隧道

```bash
cloudflared tunnel --url http://localhost:3000
```

**外网地址**: cloudflared 会显示一个 `https://xxxx.trycloudflare.com` 地址

## 方案三：使用 frp（需要公网服务器）

### 1. 准备一台公网服务器

需要有公网 IP 的服务器（如阿里云、腾讯云 VPS）

### 2. 配置 frp 服务端（公网服务器）

```ini
# frps.ini
[common]
bind_port = 7000
token = your_token
```

### 3. 配置 frp 客户端（当前服务器）

```ini
# frpc.ini
[common]
server_addr = YOUR_PUBLIC_SERVER_IP
server_port = 7000
token = your_token

[web]
type = tcp
local_ip = 127.0.0.1
local_port = 3000
remote_port = 8080
```

### 4. 启动

```bash
# 服务端
frps -c frps.ini

# 客户端
frpc -c frpc.ini
```

**外网地址**: `http://YOUR_PUBLIC_SERVER_IP:8080`

## 📝 生产环境部署建议

### 域名 + Nginx 反向代理

1. **购买域名**（可选）
2. **配置 Nginx**

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

3. **配置 HTTPS**（Let's Encrypt）

```bash
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d your-domain.com
```
