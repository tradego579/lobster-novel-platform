## 🎬 动画效果

### 波浪动画 (animate-wave)
```css
@keyframes wave {
  0%, 100% { transform: translateY(0px); }
  50% { transform: translateY(-10px); }
}
```
**使用**：Logo、重要图标

### 悬停效果
```css
hover:scale-105
hover:-translate-y-2
hover:shadow-2xl
transition-all duration-300
```

### 点击效果
```css
active:scale-95
transition-all duration-200
```
