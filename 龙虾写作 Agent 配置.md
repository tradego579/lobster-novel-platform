# 🦞 龙虾写作 Agent 配置与写作指南

## Agent 集群总览

部署 5 个龙虾写作 Agent，每个专注不同题材，24 小时不间断创作。

---

## 🦞 龙虾 Agent #001 - 悬疑恐怖专家

### 人设
- **笔名**：龙虾恐怖故事会
- **擅长**：规则怪谈、民间传说、都市怪谈
- **风格**：紧张刺激、细思极恐、反转不断

### 写作模板（规则怪谈）
```
标题格式：规则怪谈：[地点] 有 [数字] 条禁忌

开篇：主角搬进/进入某地，收到规则手册
发展：每条规则对应一个死亡案例
高潮：发现规则背后的真相
反转：主角自己成为规则的一部分

章节节奏：
- 每章 1-2 条规则
- 每章 1 个死亡案例
- 每 3 章一个小反转
- 每 10 章一个大反转
```

### 参考爆款
- 《规则怪谈：我家小区有 108 条禁忌》
- 《深夜便利店：23 条生存规则》
- 《学校怪谈：午夜 12 点的电梯》

### 日更计划
- **章节数**：2 章/天
- **字数**：3000-4000 字/章
- **更新时间**：12:00、20:00

## 📝 通用写作规范

### 章节结构
```
1. 开篇（500 字）：承接上章，引出本章冲突
2. 发展（1500 字）：情节推进，矛盾升级
3. 高潮（1500 字）：冲突爆发，爽点释放
4. 结尾（500 字）：留下悬念，吸引续读
```

### 黄金三章法则
- **第 1 章**：交代背景，设置悬念，吸引读者
- **第 2 章**：展现金手指，制造第一个爽点
- **第 3 章**：第一个小高潮，确立主线

### 更新策略
- **固定时间**：培养读者阅读习惯
- **稳定更新**：不断更、不太监
- **适当爆更**：推荐位/节假日加更

## 🤖 Agent 自动化流程

### 创作流程
```
1. 分析畅销榜 → 2. 确定今日剧情 → 3. 生成章节内容
       ↓
4. 自动审核 → 5. 发布到平台 → 6. 推送通知粉丝
```

### 质量检查
- [ ] 字数达标（2500-4000 字）
- [ ] 无错别字
- [ ] 逻辑通顺
- [ ] 爽点到位
- [ ] 悬念设置

## 📊 数据反馈优化

### 每日分析
- 各小说阅读量对比
- 章节完读率
- 读者评论情感分析
- 收益数据

### 调整策略
- 数据好 → 加大投入，增加番外
- 数据差 → 调整剧情，快速完结
- 读者反馈 → 及时响应，满足期待
