# 📚 起点/番茄风格 UI 设计文档

## 🎨 设计理念

参考起点中文网、番茄小说等主流小说平台的 UI 设计风格，打造专业、沉浸式的小说阅读和创作平台。

---

## 🌈 颜色系统

### 主色调
```
主色：橙色 #F97316 (起点风格)
辅助：红色 #EF4444 (番茄风格)
渐变：from-orange-500 to-red-500
```

### 色彩心理学
- **橙色**：活力、创作、温暖
- **红色**：热情、吸引、行动
- **白色**：简洁、阅读友好
- **灰色**：层次、专业

## 🎯 设计细节

### 按钮设计
```css
主按钮：橙色渐变 + 阴影 + hover 效果
次按钮：橙色描边 + hover 填充
尺寸：44px 高度 (最小触摸区域)
```

### 卡片设计
```css
圆角：8px
阴影：md → xl (hover)
边框：1px gray-100
背景：白色 / 深灰 (暗黑)
```

### 字体系统
```
标题：Bold, 16-24px
正文：Regular, 14px
辅助：Regular, 12px
```

### 间距系统
```
小：8px
中：16px
大：24px
超大：32px
```

## 🌙 暗黑模式

### 颜色适配
```
背景：gray-900
卡片：gray-800
文字：gray-100
边框：gray-700
主色：保持橙色
```

### 对比度要求
- 文字 ≥ 4.5:1
- 大文字 ≥ 3:1
- UI 组件 ≥ 3:1
