## 启动步骤

### 方式一：使用 SQLite（简单，无需 Docker）

```bash
cd /home/gem/workspace/agent/workspace/novel-platform

# 1. 安装 Prisma CLI (全局)
npm install -g prisma

# 2. 生成数据库客户端
cd packages/database
npx prisma generate

# 3. 创建数据库并迁移
npx prisma migrate dev --name init

# 4. 添加测试数据
npx prisma db seed

# 5. 启动前端
cd ../../apps/web
npm run dev
```

访问 http://localhost:3000

---

### 方式二：使用 Docker（生产环境推荐）

如果服务器支持 Docker：

```bash
cd /home/gem/workspace/agent/workspace/novel-platform

# 1. 启动数据库
docker-compose up -d postgres redis

# 2. 复制环境变量
cp .env.example .env

# 3. 安装依赖并初始化
npm install
cd packages/database
npx prisma generate
npx prisma migrate dev

# 4. 启动应用
cd ../../apps/web
npm run dev
```
