## 🤖 Agent 分工体系

### 1. 龙虾写作 Agent 集群（5 个）

负责小说创作，每个 Agent 专注不同题材：

| Agent | 题材 | 代表作风格 | 日更目标 |
|-------|------|------------|----------|
| 🦞#001 | 悬疑恐怖 | 规则怪谈、民间传说 | 2 章/天 |
| 🦞#002 | 都市系统 | 神豪、逆袭、打脸 | 2 章/天 |
| 🦞#003 | 科幻末世 | 重生、囤货、系统 | 2 章/天 |
| 🦞#004 | 历史商业 | 重生、创业、商战 | 2 章/天 |
| 🦞#005 | 玄幻修仙 | 无敌、苟道、升级 | 2 章/天 |

#### 写作 Agent 工作流程
```
1. 分析畅销榜 → 2. 确定选题 → 3. 生成大纲 → 4. 每日创作 → 5. 自动发布
```

#### 参考畅销思路（2026 热门）
- **规则怪谈**：小区/学校/医院禁忌，每条规则都有深意
- **国运绑定**：花钱/消费/投资变强，国家崛起
- **末世囤货**：重生 + 贷款 + 疯狂囤积，对比爽感
- **重生创业**：回到过去年代，利用信息差赚钱
- **无敌修仙**：闭关万年，出山即无敌，装逼打脸

| Agent | 职责 | 响应时间 |

#### 客服业务范围
- ✅ 账号问题（登录/注册/找回密码）

### 3. 广告业务 Agent（2 个）

负责流量变现和商业合作：

| Agent | 职责 | 目标 |
|-------|------|------|
| 🦞广告 -01 | 广告位管理、联盟对接、收益优化 | 日收益¥1000+ |
| 🦞广告 -02 | 品牌合作、IP 授权、衍生开发 | 周签约 3 家 + |

#### 广告变现渠道
1. **章节间广告**：每 3 章插入 1 个广告
2. **开屏广告**：APP/小程序启动页
3. **信息流广告**：小说列表页穿插
4. **品牌定制**：品牌方定制小说内容
5. **会员订阅**：免广告 + 抢先看

### 4. 审核 Agent（2 个）

负责内容安全和质量把控：

| Agent | 职责 | 审核标准 |
|-------|------|----------|
| 🦞审核 -01 | 内容初审（AI 自动）| 敏感词/违规内容 |
| 🦞审核 -02 | 质量复审（人工 +AI）| 逻辑/文笔/吸引力 |

#### 审核流程
```
AI 创作 → 自动初审 → 质量评分 → 通过发布 / 退回修改
```

#### 审核标准
- ✅ 无政治敏感内容
- ✅ 无色情低俗描写
- ✅ 无暴力血腥场景
- ✅ 逻辑通顺、文笔流畅
- ✅ 章节字数 2500-4000 字
